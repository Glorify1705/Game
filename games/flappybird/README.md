# Flappy Bird

A Flappy Bird clone demonstrating the engine's collision, sprite, and audio
APIs.

## Asset Attribution

Sprites and sounds are from
[samuelcust/flappy-bird-assets](https://github.com/samuelcust/flappy-bird-assets),
which recreates the assets from the original Flappy Bird game by Dong Nguyen
(.Gears / dotGears).

- **Sprites:** bird animations, pipes, backgrounds, base, UI elements, and
  digit glyphs. Packed into `flappy.qoi` with coordinates in
  `flappy.sprites.json`.
- **Sound effects:** `wing.qoa`, `point.qoa`, `hit.qoa`, `die.qoa`,
  `swoosh.qoa`. Converted from WAV to QOA (the engine's native audio format).
